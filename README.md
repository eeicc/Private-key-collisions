# 私鑰碰撞程式

本專案是一個 **私鑰碰撞程式**，透過隨機生成私鑰並轉換為公鑰，利用公鑰查詢對應的地址，當發現該地址餘額大於 0 時，會將結果記錄至 txt，要購買請聯繫我TG。

- **自動生成私鑰** 並轉換為對應的公鑰與地址。
- **查詢公鑰餘額**，發現有餘額的地址即自動記錄。
- **無錯誤運行**，可長時間穩定執行，無需人工干預。
